É um RPG de mesa brasileiro. Utiliza dados de seis lados.

# Criação de Personagem

Os personagens tem 10 pontos para gastar e podem ter até -2 pontos de desvantagens.

1. Criar o conceito;
2. [[3D&T - Atributos]];
3. [[3D&T - Perícias]];
4. [[3D&T - Vantagens]];
5. [[3D&T - Desvantagens]];

# Testes

O objetivo dos jogadores é rolar o maior numero de dados, para aumentar a chance de conseguir passar no teste.

Os jogadores SEMPRE vão rolar um dado;
Role mais um dado se tiver uma [[3D&T - Perícias]] adequada a situação;
Role mais um dado se tiver [[3D&T - Ganho]].

Uma [[3D&T - Perda]] faz o jogador rolar um dado a menos. 

Caso não exista uma pericia que faça sentido com o teste ganho o segundo dado de brinde.

# Meta

Caso um personagem tenha como resultado o dobro da meta é um critico. 

| Numero | Descrição     |
| ------ | ------------- |
| 6      | Fácil         |
| 9      | Médio         |
| 12     | Difícil       |
| 15     | Muito Difícil |
# Embates

Em caso de um personagem quanto o outro, o ativo tem vantagem no caso de empate.

Exemplo:

Um jogador esta tentando enganar um NCP, logo o jogador é o ativo e tem vantagem em caso de empate.
