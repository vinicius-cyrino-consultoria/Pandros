## Critical Success

Quanto menor for o numero melhor é o critico. 

| Soma dos dados | Quando é critico    |
| -------------- | ------------------- |
| 3              | SEMPRE              |
| 4              | SEMPRE              |
| 5              | 15 ou mais na skill |
| 6              | 16 ou mais na skill |

## Critical Failure

Quanto maior for o numero pior é o critico.

| Soma dos dados | Quando é critico    |
| -------------- | ------------------- |
| 18             | SEMPRE              |
| 17             | SEMPRE              |
Qualquer numero cuja a soma seja 10 maior que sua skill. Exemplo:
	Tirar 16 com uma skill de 6
