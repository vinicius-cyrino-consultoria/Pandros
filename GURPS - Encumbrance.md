Define o qual carregado de coisas um personagem esta. Quanto maior o nível menor o [[GURPS - Basic Move]], [[GURPS - Basic Lift]] e [[GURPS - Dodge]].

É calculada pelo [[GURPS - Basic Lift]]

| **Nível**           | **Carga (Peso)**   | **Efeito no Deslocamento (Move)** | **Penalidade na Esquiva** |
| ------------------- | ------------------ | --------------------------------- | ------------------------- |
| **0: Nenhuma**      | Até $1 \times BL$  | Move = Basic Move                 | Nenhuma                   |
| **1: Leve**         | Até $2 \times BL$  | Move = Basic Move $\times 0,8$    | -1                        |
| **2: Média**        | Até $3 \times BL$  | Move = Basic Move $\times 0,6$    | -2                        |
| **3: Pesada**       | Até $6 \times BL$  | Move = Basic Move $\times 0,4$    | -3                        |
| **4: Muito Pesada** | Até $10 \times BL$ | Move = Basic Move $\times 0,2$    | -4                        |
### **Regras Importantes**

- **Arredondamento:** Ao calcular o novo Move, ignore todas as frações (arredonde para baixo).
    
- **Mínimo Garantido:** Independentemente do peso, o Move e a Esquiva nunca podem cair abaixo de **1**.
    
- **Penalidades em Perícias:** O número do nível (0 a 4) é subtraído de testes de perícias físicas como **Escalada (Climbing)**, **Furtividade (Stealth)** e **Natação (Swimming)**.
    

### **Exemplo Prático**

Se um personagem tem **ST 10**, seu **BL é 20 lbs** (conforme a tabela da imagem):

- Se ele carregar **40 lbs**, ele está em **Carga Leve (Nível 1)**.
    
- Se o seu _Basic Move_ era **5**, o novo _Move_ será **4** ($5 \times 0,8$) e sua _Esquiva_ terá um redutor de **-1**.
